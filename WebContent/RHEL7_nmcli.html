#nmcli general status [ To show overall status of network connectivity]
#nmcli general logging [ To show network manager logging info]
#nmcli dev status [ To display the Devices List]

** Device configuration files are stored in ->
/etc/sysconfig/network-scripts/

#ifconfig ens33 [ To show particular interface details]
#nmcli connection show [ To list out the connection]

***Also we can use short form like below
#nmcli con show -a [ To list out the active connection]
#nmcli con show 'connection_name' [ To show view specific connection]

[ To add a dynamic ethernet connection ]
#nmcli con add type ethernet con-name Mycon1 ifname ens33

[ To add a static ethernet connection]
#nmcli con add type ethernet con-name Mystatic ifname ens33 ip4 192.168.1.25/24 gw4 192.168.1.1

#nmcli con down ens33 [ To down ethernet connection]
