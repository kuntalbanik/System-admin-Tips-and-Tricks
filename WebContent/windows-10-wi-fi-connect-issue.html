Windows 10 Wifi connecting issue

Open command prompt with administrator privileges and run the following commands

|

netsh winsock reset catalog

netsh int ipv4 reset reset.log

netsh int ipv6 reset reset.log

** Now disable the wifi adapter and restart the system, then connect to wifi.

** Run 'ncpa.cpl' -> right click wifi adapter -> properties -> configure -> Power Management -> uncheck the box of "Allow the computer to turn off this device to save power.

...

If same problem occurred : use the following commands

|

netsh int tcp set heuristics disabled

netsh int tcp set global autotuninglevel=disabled

netsh int tcp set global rss=enabled

...
